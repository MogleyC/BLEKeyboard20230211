# BLEKeyboard20230211

해당 제작물의 모든 소유권(소스,정보,파일,개발과정, 결과물 등)은 Mogley가 갖는다.
Mogley가 지정한 소속 관리자 및 개별 지정 담당자 외 일체의 열람/복사/사용을 금한다.
무단 열람/복사/사용이 되는 경우 해당 개인/단체에 법적 책임을 물을 수 있다.
이 외 상세사항 및 권한변경은 Mogley가 지정한 규정에 의거한다.

BLEKeyboard20230211는 주로 BLEKeyboard를 개발/관리를 위해 제작되었다.
BLEKeyboard의 주 개발 목표는 다음과 같다.
 - nRF MCU를 통한 프로젝트 개발
 - 프로젝트 회로도 및 PCB 설계 및 제작
 - 제품의 실제 구현 및 그에 필요한 작업 일체에 대한 개발 내용 정리
